# WheelView

适用 Android 开发，提供类 iOS 的滚动选择功能。

## 1. 特性

* 支持多级联动，级数自定义。
* 支持自定义 item 布局。
* 支持任何数据类型。
* 支持自定义同时显示的 item 数量。
* 支持目标数据变化的实时监听。
* 支持获取和设定当前选中项。

## 2. 文件列表

| 文件名         | 说明                   |
| -------------- | ---------------------- |
| WheelView      | 单个滚动选择控件。     |
| MultiWheelView | 多级联动滚动选择控件。 |
